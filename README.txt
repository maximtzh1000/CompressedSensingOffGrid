This is the matlab code for paper "Z. Tan, P. Yang, and A. Nehorai, "Joint sparse recovery method for compressed sensing with structured dictionary mismatch," IEEE Trans. on Signal Processing, Vol. 62, pp. 4997-5008, Oct. 2014."

Requirements: You have to add CVX to your matlab in order to run it.
For experiments results in the paper, please check subdirectories.

Author: Zhao Tan
